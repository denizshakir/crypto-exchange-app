# Crypto Exchange App

**[Live Demo](https://crypto-exchange-app-seven.vercel.app/)**

Web app providing quick access to the current market data for cryptocurrency pairs.

## Features

- Market prices
- Last Trades
- Candle chart
- Real-time order book

## Tech Stack

- React
- Vite
- Redux
- HeroUI
- lightweight-charts by TradingView
- TailwindCSS

## Installation

```bash
git clone https://github.com/denizshakir/crypto-exchange-app.git
cd crypto-exchange-app
yarn
```

## Running the app

```
yarn run dev
```
